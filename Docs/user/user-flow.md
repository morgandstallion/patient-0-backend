# User Flows

## Customer Purchase Flow

1. Customer visits website.
2. Customer browses products.
3. Customer selects a product.
4. Customer adds product to cart.
5. Customer reviews cart.
6. Customer proceeds to checkout.
7. Customer enters shipping or pickup preference.
8. Customer completes payment.
9. System creates order.
10. System sends confirmation.
11. Customer can view order status.

---

## Admin Order Management Flow

1. Admin logs in.
2. Admin views pending orders.
3. Admin prepares order.
4. Admin updates order status.
5. Customer sees updated status.

---

## Future Pickup Flow

1. Admin marks order as Ready for Pickup.
2. Customer receives notification.
3. Customer selects pickup slot.
4. System generates QR code.
5. Customer arrives at pickup site.
6. Staff scans QR code.
7. System sends OTP.
8. Customer enters OTP.
9. System verifies OTP.
10. Staff releases package.
11. Order marked as Collected.

---

## Future Delivery Flow

1. Customer selects delivery.
2. Order enters delivery queue.
3. Package assigned to delivery agent.
4. Package delivered.
5. Customer confirms receipt.
6. Order marked as Delivered.
